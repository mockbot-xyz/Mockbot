ANSV Bot

Markov chain text generator for Twitch chat. Free, with optional $2/mo TTS.

Sign in with Twitch

What it does

Learns word patterns from your chat and generates new messages. Uses a Markov chain - a statistical model that predicts the next word based on previous words.

Example:

Training:
viewer1: lol this game is so bad
viewer2: the game is pretty fun actually
viewer3: i love this game PogChamp

Output:
ANSV_Bot: this game is so fun lmao
ANSV_Bot: lol i love this is pretty fun PogChamp

Output is often grammatically wrong - that's normal.

How it works

  1. Bot connects to your Twitch chat via IRC
  2. Stores messages in SQLite
  3. Builds probability tables with markovify
  4. Generates and posts messages

Uses bigram/trigram models. Needs 1000+ messages for decent output.

Features

  • Markov text generation
  • Auto-post or command-triggered
  • Web dashboard
  • Configurable frequency/intervals
  • TTS with Bark model (Premium)
