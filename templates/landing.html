What is a Markov Chain?

A Markov chain is a statistical model that predicts the next word in a sequence based on the previous words. The bot analyzes your chat history to learn which words commonly follow each other, then generates new messages by probabilistically choosing the next word based on those patterns.

Training Data (What the bot learns from)

viewer1: lol this game is so bad
viewer2: the game is pretty fun actually
viewer3: i love this game PogChamp
viewer4: this is so fun lmao

What the Bot Generates

ANSV_Bot: this game is so fun lmao
ANSV_Bot: lol i love this is pretty fun PogChamp

The bot learned phrases like "this game is" and "so fun" from the training data, then combined them probabilistically. Sometimes the output is grammatically incorrect or nonsensical - that's normal for Markov chains.

How It Works

  1. Bot joins your channel - ANSV connects to your Twitch chat via IRC
  2. Collects messages - Stores all chat messages in a SQLite database
  3. Builds the chain - Analyzes word patterns and builds probability tables
  4. Generates text - Walks through the chain to create new messages
  5. Posts to chat - Sends generated messages at intervals you configure
Technical note: This uses a bigram/trigram model with markovify. The more chat data, the better the output. Channels with <1000 messages will produce mostly nonsense.

Features
