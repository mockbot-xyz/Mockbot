About Health Monitoring

Connection Status: Shows whether the bot is connected to Twitch. Automatic reconnection attempts will be displayed in real-time.

Errors: Tracks system errors over the last hour and 24 hours. Critical errors trigger browser notifications.

Performance: Real-time CPU and memory usage of the bot process.

Background Tasks: Health status of heartbeat and message request checker tasks.

Auto-Updates: This dashboard updates automatically via WebSocket and polls every 30 seconds as a fallback.
