# Welcome to Mockbot

Welcome to the complete guide for Mockbot. This documentation covers all of the bot's core features, settings, and its Custom Command & Generative Grammar engine.

## What is Mockbot?

Mockbot is a Markov-chain based Twitch chatbot that learns how your chat speaks and generates dynamic responses. It includes integrations for native Twitch Polls, timers, Text-to-Speech (TTS), and a custom command engine powered by Funtoon Grammar.
